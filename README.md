# Sistem Prediksi Kelayakan Bantuan Sosial

Aplikasi web berbasis Flask untuk memprediksi kelayakan penerima bantuan sosial
menggunakan metode Logistic Regression dan Random Forest.

## Fitur
- Input data calon penerima bantuan
- Prediksi kelayakan bantuan sosial
- Evaluasi model dengan Confusion Matrix dan ROC Curve
- Perbandingan model Logistic Regression dan Random Forest
- Visualisasi Feature Importance dan distribusi data

## Teknologi
- Python
- Flask
- Scikit-learn
- Matplotlib & Seaborn
- Bootstrap

## Cara Menjalankan
```bash
pip install -r requirements.txt
python app.py
